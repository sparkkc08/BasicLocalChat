# Basic Local Chat

Простейшая реализация локального чата с использованием WebSocket

Для работы необходимо 2 устройства, одно из которых будет выполнять роль сервера.

1. Запустить приложения на двух устройствах.
2. Ввести имена(псевдонимы).
3. На одном устройстве нажать кнопку "Создать чат". На остальных устройствах "Войти в чат",и на следующем экране поключиться к серверу, введя IP адрес, который отображается на экране чата устройства-сервера.
4. Писать сообщения в спецаильном поле. Для отправки нажать кнопку с иконкой конверта.
5. Для просмотра списка пользователей сделать скрол с левого края экрана или нажать на кнопку меню, которая расположена сслева вверху.
6. Кликнуть на имя нужного пользователя для отправки приватного сообщения.
7. Для возвращения к экрану входа воспользоваться меню действий справа вверху.
